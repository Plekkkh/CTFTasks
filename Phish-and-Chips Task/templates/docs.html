Internal Architecture Note (v2)

⚠️ Notice to Developers & QA:

The legacy URL scanner (v1) is available via public REST endpoints.

The new Advanced Engine (v2) is currently running on 127.0.0.1:5000 for internal integration tests. It is strictly firewalled and must not be exposed to the external network until the security audit is complete!

Internal Testing Usage

For internal testing, send GET requests to the microservice directly via the internal proxy:

GET /api/v2/analyze?target=[URL] HTTP/1.1
Host: 127.0.0.1:5000
