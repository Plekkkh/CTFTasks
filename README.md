Добро пожаловать в мой репозиторий авторских CTF-задач!

Здесь собраны таски, которые я разрабатываю для проектного практикума и соревнований ИРИТ-РТФ УрФУ на направлении "Информационная безопасность". 

Моя цель — создавать интересные и разнообразные задачи, которые помогут участникам развивать навыки в различных областях ИБ, таких как веб, реверс-инжиниринг, криптография и многое другое.

Репозиторий служит портфолио моих навыков в области уязвимостей, инфраструктурного пентеста и разработки.

Каждая задача изолирована, упакована в Docker (по возможности) и содержит подробное описание и инструкции по запуску.

## Как использовать этот репозиторий

Вам не нужно устанавливать зависимости локально. Вся нужная инфраструктура поднимается через контейнеры.

1. Склонируйте репозиторий на свою машину:
   ```bash
   git clone https://github.com/JeekaMIX/CTFTasks.git
   ```
2. Перейдите в директорию выбранной задачи (например, `cd CTFTasks/Phish-and-Chips`).
3. Откройте локальный `README.md` внутри папки для чтения лора и инструкций по запуску.
